# Фронтенд (Vue 3 + Vite)

Для фронтенда используется только `npm` (`frontend/package-lock.json` — источник истины).
`yarn` и `pnpm` в этом репозитории не поддерживаются.

## Локальный запуск

```bash
cd frontend
npm ci
npm run dev
```

## Проверки

```bash
cd frontend
npm run lint
npm run build
```

## Автоисправление lint

```bash
cd frontend
npm run lint:fix
```
